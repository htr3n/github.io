Spring Data JPA

Nov 22, 2018 in DEV
spring framework database jpa spring boot transaction
1 min read

Java Persistence API (JPA) is historically part of the Java EE platform. It defines a standard Java API dealing with relational databases. EclipseLink (formerly Oracle TopLink) is considered a reference implementation but Red Hat (formerly JBoss) Hibernate ORM has significantly influenced the current shape of JPA. Apart from EclipseLink and Hibernate, there are also some other JPA implementations such as DataNucleus, Apache OpenJPA.

A part of Spring framework, namely, Spring Data JPA, is sort of a façade that provides a data access abstraction layer rather than a true JPA implementation. We can choose any JPA providers, not only Hibernate or EclipseLink but also anything else.

  • PagingAndSortingRepository

  • CrudRepository
