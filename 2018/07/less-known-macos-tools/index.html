Disk Utility

Resize
hdiutil resize -size 10g filename.sparsebundle
Creating a sparsebundle (rw)
hdiutil create -size <size>g -fs HFS+J -volname <volume_name> <file-name>.sparsebundle
hdiutil resize -limits rw.dmg
hdiutil create -verbose -type SPARSEBUNDLE -fs HFS+J -encryption AES-128 -stdinpass -nospotlight -volname "HOME" -size 200g HOME.sparsebundle
Creating ISO image from a path or CDR
hdiutil makehybrid -o image.iso ~/path/to/folder -iso -joliet
hdiutil makehybrid -iso -joliet -o [filename].iso [filename].cdr
Convert DMG RO <–> RW
hdiutil convert ro.dmg -format UDRW -o rw.dmg
hdiutil convert rw.dmg -format UDZO -imagekey zlib-level=9 -o ro.dmg
Options
  • UDRW - UDIF read/write image
  • UDRO - UDIF read-only image
  • UDCO - UDIF ADC-compressed image
  • UDZO - UDIF zlib-compressed image
  • UDBZ - UDIF bzip2-compressed image (OS X 10.4+ only)
  • UFBI - UDIF entire image with MD5 checksum
  • UDRo - UDIF read-only (obsolete format)
  • UDCo - UDIF compressed (obsolete format)
  • UDTO - DVD/CD-R master for export
  • UDxx - UDIF stub image
  • UDSP - SPARSE (grows with content)
  • UDSB - SPARSEBUNDLE (grows with content; bundle-backed)
  • RdWr - NDIF read/write image (deprecated)
  • Rdxx - NDIF read-only image (Disk Copy 6.3.3 format)
  • ROCo - NDIF compressed image (deprecated)
  • Rken - NDIF compressed (obsolete format)
  • DC42 - Disk Copy 4.2 image
