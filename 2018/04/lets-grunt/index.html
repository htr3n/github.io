Let's Grunt

Apr 26, 2018 in DEV
javascript node.js grunt task-runner automation
1 min read

Grunt is a JavaScript/Node.js based task runner in some ways very similar to Makefile and Apache Ant. It requires Node.js beforehand, just like Ant needs a Java Runtime. After installing Node.js, Grunt command line can be installed as following.

npm install -g grunt-cli

Grunt tasks are defined in a configuration Gruntfile.js (JavaScript) or Gruntfile.coffee (CoffeeScript).

// Grunt's wrapper function
module.exports = function(grunt) {
	// let's grunt in here
};

Grunt plugins and tasks

grunt.loadNpmTasks('grunt-contrib-uglify');
1. A Simple Task
// a simple greeting task
module.exports = function (grunt) {
  grunt.registerTask('default', 'Simple Hello', function () {
    grunt.log.write('Hello World from Grunt!').ok();
  });
};
