hyde-hyde

A simple and neat Hugo theme

New

hyde-hyde is a responsive Hugo’s theme derived from @spf13’s Hyde which is in turn ported from @mdo Jekyll’s Hyde. It was heavily restructured with modularised page layouts for easier maitenance and modification. The color tone is inspired by the super cool Nate Finch’s blog. hyde-hyde also incorporates many other extra features such as code highlighting (using highlight.js), nice sidebar icons (using Font-Awesome 5), with Fira-Sans as main font and Roboto Mono as fixed width font, and another choice for commenting with GraphComment besides the built-in Disqus.
