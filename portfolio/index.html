hyde-hyde

A simple and neat Hugo theme

New

hyde-hyde is a responsive Hugo’s theme derived from @spf13’s Hyde. It was heavily restructured with modularised page layouts for easier maitenance and modification. The color tone is inspired by the cool Nate Finch’s blog. hyde-hyde incorporate some extra features such as code highlighting (using highlight.js), nice sidebar icons (using Font-Awesome 5), with Fira-Sans as main font and Roboto Mono as fixed width font, and another choice for commenting with GraphComment besides the built-in Disqus.

DERA

Dynamic Event-Driven Actors

This is an implementation of DERA system using Java (core) and Groovy (examples). Dynamic Event-driven Actors (DERA) is an approach aiming at providing adequate means for modeling and developing (distributed) event-based systems. DERA leverages the intrinsic loose coupling of event-driven communication styles to support various kinds of run-time evolution and adaptation (i.e., enabling run-time flexibility) while minimizing the non-deterministic nature of traditional event-based applications (for supporting formal analysis).

LaraMod

A modularised Laravel framework

LaraMod is another modularisation effort to organising a Laravel based project. The idea stems from my struggle to organise a Laravel-based project such that I can work effectively and focus better on individual modules whilst keeping Laravel codebase intact as much as possible (this would be convenient for upgrading Laravel) and also keeping my codebase separate from the original Laravel. Besides, LaraMod embraces Twitter’s Bootstrap 3 and the beautiful Gentelella Admin.
